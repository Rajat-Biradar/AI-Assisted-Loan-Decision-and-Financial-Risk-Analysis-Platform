# AI-Assisted Loan Decision and Financial Risk Analysis Platform

## Overview

AI-Assisted Loan Decision and Financial Risk Analysis Platform is a web-based application designed to help users evaluate loan affordability, analyze financial risk, and make informed borrowing decisions.

The platform combines traditional financial calculations with Machine Learning-based prediction to provide EMI analysis, loan eligibility estimation, bank recommendations, repayment planning, and financial burden assessment through a single interactive dashboard.

---

## Features

* EMI Calculation and Repayment Analysis
* Debt-to-Income (DTI) Risk Assessment
* Loan Eligibility Estimation
* Bank Recommendation System
* Financial Burden Prediction using Machine Learning
* Early Repayment Simulation
* Interactive Charts and Financial Analytics
* Responsive Web Interface

## System Workflow

1. User enters loan and financial information.
2. System calculates EMI and repayment details.
3. DTI ratio is computed for risk assessment.
4. Loan eligibility is estimated.
5. Machine Learning model predicts financial burden score.
6. Banks are compared and ranked.
7. Results are displayed through charts and analytics dashboards.

```bash

```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
